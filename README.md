# Calculatrice

Une application web simple et conviviale qui permet aux utilisateurs de réaliser des opérations de base telles que l'addition, la soustraction, la multiplication et la division.
Développée en utilisant HTML, CSS et JavaScript.

Les fonctionnalités clés de cette calculatrice incluent :

- Une interface utilisateur conviviale et intuitive
- Une entrée de texte pour saisir les nombres et les opérations
- Une fonction de calcul automatique qui affiche le résultat de l'opération
- Des boutons pour les opérations de base (addition, soustraction, multiplication, division)
- Une fonction de nettoyage qui permet de réinitialiser l'affichage et de commencer à zéro

Pour utiliser la calculatrice, il suffit de charger la page web et de saisir les nombres et les opérations dans les champs appropriés.
Ensuite, appuyez sur le bouton correspondant à l'opération souhaitée pour obtenir le résultat.
